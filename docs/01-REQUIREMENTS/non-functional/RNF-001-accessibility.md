# RNF-001: Accessibility

## Requirement
The mobile app must be accessible to users with limited technical literacy.

## Acceptance Criteria
- Touch targets minimum 48x48dp (WCAG 2.5.5)
- Button labels 16sp minimum, 24sp recommended
- High contrast color ratios (AA minimum)
- Large numeric keypad for PIN entry
- Clear visual feedback for all actions

## Priority
Must-have
