# RF-001: Clock-in / Clock-out

## Requirement
Workers must be able to register clock-in, clock-out, lunch start, and lunch end events.

## Acceptance Criteria
- Four action buttons displayed on home screen
- Each button has distinct color, icon, and Portuguese label
- Tapping a button navigates to QR scan screen
- Time log is recorded with worker_id, boss_id, log_type, logged_at, qr_code_value

## Priority
Must-have
