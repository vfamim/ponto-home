# Project Charter

## Project Name
Ponto Home

## Purpose
Android time-tracking application for domestic staff with a boss dashboard for monitoring.

## Scope
- Mobile app (Expo/React Native): Worker clock-in/out via QR scan + selfie
- Web dashboard (Vite/React): Boss manages workers, views time logs, exports CSV
- Backend (Supabase): Auth, database, storage, edge functions

## Success Criteria
- Workers can clock in/out with 2 taps (action + selfie)
- Boss can register workers, view real-time status, export time logs
- Offline support with automatic sync
- APK distribution (no app store)

## Timeline
Phase 1-5: Foundation through CI/CD
Phase 6: Release v0.1.0
