# QR Code Physical Specifications

| Parameter | Specification |
|-----------|--------------|
| QR Version | Version 3 (29x29 modules) |
| Error Correction | Level H (30% recovery) |
| Printed Size | 15cm x 15cm (minimum 10cm) |
| Module Scaling | 4mm per module at 15cm |
| Quiet Zone | 4-module white border (1.6cm min) |
| Print Material | Matte laminated vinyl or PVC foam board |
| Lamination | Anti-reflective matte overlaminate |
| Phosphorescent Border | Glow-in-the-dark strip (1cm wide) |
| Mounting Height | 140cm from floor (center) |
| Mounting Location | Wall adjacent to kitchen entrance, left side |
| Mounting Method | Adhesive + 4 corner screws on PVC board |
| Background | Pure white (#FFFFFF) |
| Foreground | Pure black (#000000) |
