# APK Install Procedure

## Prerequisites
- Android device running Android 8.0+ (API 26+)
- "Install from unknown sources" enabled in device settings

## Steps

1. Transfer APK to device
   - Via USB: Copy APK file to Downloads folder
   - Via link: Boss shares download link from EAS Build

2. Enable unknown sources
   - Settings > Security > "Install unknown apps" > Enable for browser/file manager

3. Install APK
   - Open file manager, navigate to APK
   - Tap APK, confirm installation

4. Launch app
   - Open from app drawer or home screen
   - Grant camera permission when prompted

5. First use
   - Worker enters 4-digit PIN assigned by boss
   - Home screen displays action buttons

## Updating
- Download new APK and install over existing version
- Data is preserved automatically
